T1  文本字体    | cmss 现代等线体  | bx粗宽 | it斜体      | 5~25pt
OML 数学倾斜体  | cmtt 现代打字机体 |       | s1倾斜      |
OMS 数学符号    | cmm  现代数学斜体 |       | sc小型大写  |
OMX 数学大符号  | cmsy 现代数学符号 |       |            |
U   未知       | cmex 现代数学扩充 |       |            |
